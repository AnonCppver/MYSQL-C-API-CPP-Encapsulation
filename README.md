# MYSQL-C-API-CPP-Encapsulation
这是MYSQL C API的C++封装代码，目的是简化MYSQL C++开发流程，包含了connection和preparedstatement两个类来支持更简洁预处理语句。This project is a C++ wrapper for the MySQL C API, designed to simplify MySQL development in C++.It provides two core classes — Connection and PreparedStatement — enabling a cleaner and more intuitive workflow when working with MySQL databases.
